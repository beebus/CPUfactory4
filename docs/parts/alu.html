Arithmetic-logic Unit¶

This part is the ALU unit. IT performs both 8-bit and 16-bit operations

Pin Def Description
1 op1 16-bit operand 1
2 op2 16-bit operand 2
3 aop 4-bit selector
4 res 16-bit result
5 Z Zero status flag
